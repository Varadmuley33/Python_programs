# Python Programs Collection 🐍

This repository contains a collection of **103 Python programs** that I created while learning Python — starting from the most basic concepts and gradually progressing to advanced topics, including data structures.

## 📚 Learning Journey

- **Getting Started** – Simple Python syntax, printing output, taking user input.
- **Basic Concepts** – Variables, data types, type casting, operators, and conditional statements.
- **Control Flow** – If-else statements, loops (`for`, `while`), nested loops.
- **Functions** – Defining and calling functions, parameters, return values.
- **Strings** – String operations, slicing, formatting, and built-in string methods.
- **Lists & Tuples** – Creating, updating, iterating, and using list/tuple methods.
- **Dictionaries & Sets** – Key-value pairs, set operations, and related methods.
- **File Handling** – Reading from and writing to files.
- **Modules & Libraries** – Importing and using built-in Python modules.
- **Data Structures** – Implementations of stacks, queues, linked lists, and other structures.

## 🛠 Technologies Used
- **Language:** Python 3.x
